## 開発環境構築手順
1. GitHubからリポジトリをクローン
   ```sh
   git clone https://github.com/yuitov0o0v/fenrir_kadai.git
   cd restaurant_search_app
   ```
2. 必要なパッケージをインストール
   ```sh
   bundle install
   ```
3. envファイルの作成
   ```sh
   touch .env
   echo "HOTPEPPER_API_KEY=(あなたのAPI KRYを入力)" >> .env
   ```
5. データベースをセットアップ
   ```sh
   rails db:create
   rails db:migrate
   ```
6. サーバーを起動
   ```sh
   rails server
   ```
7. ブラウザで `http://localhost:3000` にアクセス
